# ROG_S17 Interview

- 使用 Node.js + Express + Mongodb 打造的餐廳美食網站，在此專案能夠利用搜尋功能來尋找餐廳。
- 並且能夠新增、修改、刪除餐廳的詳細資訊至資料庫。
- 能夠依照餐廳名稱、類別、分數來排列順序

![index-image](https://imgur.com/3LfPrDp)

## 啟動方式

- 將專案 clone 到本地端

```
https://github.com/ashley5233/ROG_S17.git
```

- 進入到專案資料夾

```
cd ROG_17
```

- 安裝 npm

```
$npm install
```
